# Lulla SEO Cloudflare release v1

Deploy this folder to Cloudflare Pages.

Temporary canonical base: `https://lulla-wake-window.pages.dev`

Before deployment:
1. Confirm the Pages project name. If it differs, replace `https://lulla-wake-window.pages.dev` in all HTML, `robots.txt`, and `sitemap.xml`.
2. Replace placeholder legal/contact copy in Privacy and Terms.
3. Test all calculator actions on the deployed HTTPS URL.
4. Add the property to Google Search Console and submit `/sitemap.xml`.
5. Consider Cloudflare Web Analytics, without sending exact baby sleep inputs.

The root page targets the main tool intent. Static supporting pages target age-specific wake-window and nap-length searches.
